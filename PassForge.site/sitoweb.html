Chi siamo

Siamo tre studenti dell'ITIS'Grassi', iscritti all’indirizzo di Informatica, attualmente frequentiamo la classe 1SINF1. Siamo appassionati di tecnologia e programmazione, e ci impegniamo a sviluppare nuove competenze nel campo dell'informatica.

Funzionalità dell'app

Passforge offre una generazione di password sicure, personalizzabili in base a lunghezza e complessità. Ogni password è unica e crittografata per garantire la massima sicurezza.

Basi dell'applicazione

Nel nostro progetto, abbiamo combinato diverse tecnologie per creare una soluzione sicura ed efficiente. Abbiamo utilizzato Python per la parte logica, HTML e CSS per l'interfaccia utente, e implementato algoritmi avanzati per generare password che soddisfano i più elevati standard di sicurezza.
