Funzionalita dell app

Interfaccia

L'interfaccia di questo generatore di password è progettata per essere intuitiva e facile da usare, permettendo all'utente di creare password sicure in pochi passi.

Crittografia

Tutte le credenziali salvate vengono cittografate tramite la libreria Crptography di python, quindi tutte le password saranno al sicuro.

Creazione di Passforge

Il programma è stato fatto con il linguaggio python ed è completamente open source. La parte grafica è stata fatta con tkinter.

Master Password

La prima volta che aprirete il programma vi sarà chiesta una masterpassword,questa serve a decifrare le password che salverete.

Lista delle password

Le password salvate si possono vedere con il pulsante "Mostra password", apparirà questa finestra.

Eliminazione delle password salvate.

Per eliminare le password bisogna scrivere il nome del servizio che si vuole eliminare e schiacciare il tasto elimina, per sicurezza comparirà una finestra che ti chiederà se sei sicuro di volerla cancellare.
